       PROJET - Partie2.C - sujets complémentaires proposés - 

Choix proposés : 
• l'objectif des choix C1 à C4 est de réaliser une transformation entre certains formats :
     avec  XQUERY (choix C1 ou C2)
      ou avec PHP (choix C3 ou C4)  
(par exemple pour générer un site web à partir de données XML) ; 
• le choix C5 concerne RDF.

Modalités :
• il s'agit d'un complément au choix (C1 ou C2 ou C3 ou C4 ou C5) et selon le temps disponible (si les parties 2.A et 2.B sont complètement terminées) ;

• pour cette partie 2.C : il n'y aura pas de pénalité si elle est non traitée, quelques points (peu) lui étant attribuée dans le barème du projet.  



Remarques. 
• En 2023, une introduction à RDF et à XQUERY a été vue en CM ; PHP n'a pas été vu en DSB cette année, mais des documents d'années antérieures sont fournis (à étudier par vous-même, pour les choix C3 ou C4). 
•  D'autres langages pourraient jouer le rôle de XQUERY ou de PHP, avec le même objectif ; si certains étudiants souhaitent utiliser un autre langage, en discuter avant avec l'équique pédagogique. 
------------------------------------------------------------------------

• C1. XQUERY, XML vers XHTML 
Etudier les documents fournis sur XQUERY. 
Réaliser une requête XQUERY pour transformer  votre fichier XML en page(s) web. Rendre le résultat accessible depuis votre site de projet. 
------------------------------------------------------------------------

• C2. XQUERY, XML vers SQL
Etudier les documents fournis sur XQUERY. 
Réaliser une requête XQUERY pour transformer  le format de votre fichier XML de façon à insérer des données  dans votre base MySQL.
------------------------------------------------------------------------

• C3. PHP,    XML vers XHTML 
Etudier les documents fournis sur PHP (à étudier par vous-même, PHP n'a pas été vu en DSB cette année). 
Réaliser un programme PHP pour transformer  votre fichier XML (en partie 2) en page(s) web. 
Rendre le résultat accessible depuis votre site de projet. 
------------------------------------------------------------------------

• C4. PHP, SQL vers XHTML
Etudier les documents fournis sur PHP (à étudier par vous-même, PHP n'a pas été vu en DSB cette année). 
Réaliser un programme PHP pour transformer une table SQL (ou une requête SQL en partie 1) en une page du site web.  
Rendre le résultat accessible depuis votre site de projet. 

-- en détail : 
Donnez un code PHP permettant :
    • de se connecter à une base de données 
    • d’extraire les n-uplets de la table ou de la requête
    • d’afficher ces n-uplets dans un tableau HTML ayant pour première ligne le nom des attributs
et pour aller plus loin : 
    • faire en sorte que le contenu d’une des colonnes soit un lien hypertexte (fictif) dont la destination doit changer selon le contenu de la colonne (par exemple, cliquer sur la valeur 51 du tableau ACTEUR doit envoyer vers la page fictive http://www.allocine.fr/?filmid=51).
------------------------------------------------------------------------
C5. RDF
Dessinez un graphe de données RDF représentant quelques données de vos tables 
(voir le CM et le document de TD, avec sa correction).
Donnez une traduction RDF (notation Turtle) de votre graphe. 
Vous pouvez inventer des préfixes utiles (fictifs) si besoin.  
Rendre le résultat accessible depuis votre site de projet.

Description des entités

Catégorie

Attributs Type Taille Description
id_cat* INT 20 Clé primaire
lib_cat VARCHAR 100 Libellé de la catégorie

Produit

Attributs Type Taille Description
id_prod* INT 20 Clé primaire
nom_prod VARCHAR 100 Nom du produit
descri_prod VARCHAR 255 Description du produit
prix_achat_prod DOUBLE Prix d’achat HT du produit
prix_vente_prodt DOUBLE Prix de vente HT du produit
qte_stock INT 10 Quantité en stock
date_peremption DATE Prix de vente HT du produit
Date de péremption DATE Date d’approvisionnement la plus récente

Client

Attributs Type Taille Description
id_client* INT 20 Clé primaire
nom_client VARCHAR 40 nom du client
prenom_client* VARCHAR 40 Prénom du client
sexe_client VARCHAR 1 M (Masculin) ou F (Féminin)

Fournisseur

Attributs Type Taille Description
id_fourn* INT 20 Clé primaire
nom_fourn VARCHAR 40 nom du fournisseur
prenom_fourn* VARCHAR 40 Prénom du client
sexe_fourn VARCHAR 1 M (Masculin) ou F (Féminin)
contact_fourn* VARCHAR 15 Numéro de téléphone du fournisseur
mail_fourn VARCHAR 15 Adresse mail du fournisseur

Promotion

Attributs Type Taille Description
id_cat* INT 20 Clé primaire
lib_promo VARCHAR 100 Libellé de la promotion
pourcentage_reduction DOUBLE Pourcentage de réduction sur les articles concernés par la promotion
date_debut DATE Date de début de la promotion
date_fin DATE Date de fin de la promotion

Don

Attributs Type Taille Description
id_don* INT 20 Clé primaire
lib_don VARCHAR 100 Libellé du don
date_reception DATE Date de réception du don par une association

Association

Attributs Type Taille Description
id_asso* INT 20 Clé primaire
nom_asso VARCHAR 100 Nom de l’association
